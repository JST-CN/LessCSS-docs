> 可以在你的样式中使用的函数。

## 生成颜色(Generating colors)

```less
rgb(r, g, b)
```

```less
rgba(r, g, b, a)
```

```less
hsl(h, s, l)
```

```less
hsla(h, s, l, a)
```

## 修改颜色(Modifying colors)

```less
saturate(color, amount)
```

```less
desaturate(color, amount)
```

```less
lighten(color, amount)
```

```less
darken(color, amount)
```

```less
greyscale(color, amount)
```
